# grafana

## Instalación

Hay que hacerse una cuenta en https://grafana.com/ 

Añadir un `stack` si no lo tiene creado. La versión gratis solo puede tener uno.

Entrar al entorno de grafana para ese stack, dando click en `Launch`.
